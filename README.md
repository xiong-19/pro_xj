# 新疆自驾游行程规划器

一个基于 Web 的新疆自驾游行程规划工具，支持地图标记、路线绘制、行程管理等丰富功能。

## 🌟 功能特性

### 核心功能
- ✅ **实时自动保存** - 所有修改自动保存到浏览器 localStorage
- ✅ **地图标记** - 支持搜索地点和手动选点两种方式
- ✅ **路线绘制** - 在地图上绘制每日行程路线
- ✅ **行程管理** - 14 天行程详细规划（景点、美食、住宿）
- ✅ **多媒体标记** - 支持图片、文本标注
- ✅ **数据备份** - 手动导出/导入 JSON 备份文件

### 地图工具
- 🔍 **搜索地点** - 搜索并标记地点（水滴状图标）
- 🛣️ **绘制路线** - 两点间自动规划路线
- 📍 **手动选点** - 点击地图任意位置添加标记
- 📝 **添加文本** - 在地图上添加文字标注
- 📷 **添加图片** - 在地图上添加图片标记
- ❌ **删除模式** - 删除不需要的标记
- 🔄 **重置地图** - 恢复地图初始状态

### 数据存储
- **localStorage** - 浏览器本地存储，实时自动保存
- **手动备份** - 导出 JSON 文件到本地
- **导入恢复** - 从备份文件恢复数据
- **跨会话** - 关闭浏览器后数据不丢失

## 📖 使用指南

### 第一次使用

1. **打开应用** - 双击 HTML 文件或访问 GitHub Pages 链接
2. **切换到编辑模式** - 点击右上角"✏️ 修改"按钮
3. **开始规划** - 使用地图工具添加地点和路线

### 添加地点

#### 方法 1：搜索地点
1. 点击"🔍 搜索地点"
2. 输入地点名称（如：喀纳斯湖）
3. 输入标记名称（可选）
4. 点击搜索，自动标记到地图

#### 方法 2：手动选点
1. 点击"📍 手动选点"
2. 填写地点信息：
   - 地点名称（必填）
   - 标记颜色（6 色可选）
   - 地点描述（可选）
   - 关联日期（可选）
3. 点击"确认添加"
4. 在地图上点击想要标记的位置

### 绘制路线
1. 点击"🛣️ 绘制路线"
2. 选择日期
3. 选择起点和终点
4. 点击"确认绘制"
5. 在地图上点击设置起点和终点

### 管理行程
1. 点击底部日期卡片
2. 编辑行程信息：
   - 路线安排
   - 景点游览
   - 美食推荐
   - 住宿信息
   - 行驶距离
   - 其他备注

### 数据备份

#### 导出备份
- 点击"💾 备份"按钮
- 浏览器会下载 JSON 备份文件
- 建议定期备份（每天结束时）

#### 导入恢复
- 点击"📂 导入"按钮
- 选择之前备份的 JSON 文件
- 数据会自动恢复

### 删除元素
1. 点击"❌ 取消工具"（进入删除模式）
2. 点击要删除的标记或路线
3. 再次点击退出删除模式

## 💾 数据存储说明

### localStorage 存储
- **位置**：浏览器本地存储
- **容量**：约 5-10MB
- **持久性**：除非清空浏览器数据，否则永久保存
- **自动保存**：所有修改实时保存，无需手动操作

### 备份文件
- **格式**：JSON
- **命名**：`xinjiang_travel_backup_YYYY-MM-DD_timestamp.json`
- **内容**：完整的行程数据（日期、地图标记、路线）

### 数据加载优先级
```
1. localStorage（最快，优先）
2. 数据文件（xinjiang_travel_data.json）
3. 空数据（新建）
```

## 📝 常见问题

### Q: 数据会丢失吗？
A: 不会。所有修改都会实时保存到 localStorage。即使关闭浏览器，下次打开也会自动恢复。建议定期手动备份。

### Q: 可以跨设备使用吗？
A: localStorage 数据仅限当前设备。如需跨设备，可以使用备份/导入功能：
1. 在设备 A 导出备份
2. 将备份文件发送到设备 B
3. 在设备 B 导入备份

### Q: 为什么不能自动保存文件？
A: 这是浏览器的安全限制。网页不能直接修改本地文件，需要用户授权。我们使用 localStorage 实现自动保存，手动备份用于数据迁移。

### Q: 地图加载慢怎么办？
A: 使用高德地图需要国内网络环境。如果加载慢，可以：
1. 检查网络连接
2. 刷新页面
3. 等待地图缓存

### Q: 如何清空所有数据？
A: 点击"🗑️ 清空"按钮，确认后即可清空所有数据。操作不可恢复，请谨慎使用。
