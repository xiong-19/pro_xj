📖 如何获取 Token？

  1. 访问 GitHub Token 设置页面
  2. 点击 "Generate new token (classic)"
  3. Note：填写任意内容（如：Xinjiang Travel）
  4. Expiration：建议 30 天
  5. 勾选 repo 权限（必须）
  6. 点击 "Generate token"
  7. 立即复制 Token（只显示一次）

💡 使用说明

  • Token 会保存在本地浏览器，不会上传到 GitHub
  • 每次同步数据时会使用此 Token
  • 可以随时更换 Token
  • 如果 Token 失效，重新生成后在此更新
